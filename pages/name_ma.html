엑셀 2파일 매칭 & 출력 데이터 생성

0) 처리 개요

- 이름 매칭은 공백 제거 기준으로 수행합니다.
- 결과 파일에서도 이름 컬럼 값은 공백 제거 후 출력됩니다.
- 디폴트 헤더(파일1 전체 컬럼)는 헤더 고정(출처/컬럼 변경해도 헤더명 자동 변경 없음).
- 다운로드 파일명은 버튼명과 동일하게 저장됩니다.
- 출력 헤더명이 중복되어도 각 매핑별 열로 유지됩니다(덮어쓰기 방지).

3) 출력 헤더(만들 데이터) 구성

- 헤더명을 수정하지 않으면 원본 컬럼명을 자동 사용합니다.
- 디폴트 헤더(파일1 전체 컬럼) 생성 시, 헤더가 고정됩니다.
- 출처=상수일 때, 프리셋 선택 또는 직접 입력이 가능합니다.
※ 디폴트 헤더는 현재 매핑을 덮어씁니다.
